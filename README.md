# 🏠 FatCat Cafe — FF14 RP Staff Showcase

A static web page for the *Final Fantasy XIV* RP venue **FatCat Cafe** (Meteor · Red Tea River · Ul'dah 23-19), displaying employee profiles, group photos, posters, and the menu.

## File Structure

```
ff14-project/
├── index.html                    # Main page (open to browse)
├── README.md                     # This file
├── data/
│   ├── staff.md                  # Raw staff data
│   ├── staff/
│   │   ├── staff_01.md           # Individual staff profiles
│   │   └── ...
│   └── menu.md                   # Menu text content
└── images/
    ├── cards/                    # Staff card images (card_01–card_39)
    ├── groups/                   # Group photos
    ├── posters/                  # Promotional posters
    └── menu.jpg                  # Menu image
```

## Usage

Open `index.html` in any browser. No server required.

- Click a staff card to view the full profile.
- Click a group photo, poster, or the menu image to view it enlarged.
