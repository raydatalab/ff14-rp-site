# FF14 RP Staff Page

A static HTML page showcasing staff, group photos, posters, and menu for a FF14 RP shop.

## Structure

- `index.html` — main page
- `data/` — staff and menu data
- `images/` — all image assets

## Card Specs

- **Image format**: WebP
- **Card dimensions**: 2076 × 974 px (landscape)
- **Display preference**: modal top-bottom layout for landscape images

## Usage

Open `index.html` in any browser.
