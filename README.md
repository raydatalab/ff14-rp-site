# 🏠 肥肥之家 — FF14 RP 店员展示页面

> 豆豆柴 · 红茶川 · 乌尔达哈 23-19

基于《最终幻想14》RP 店「肥肥之家」的员工与菜单信息，生成的静态展示页面。

---

## 📖 项目简介

本项目将肥肥之家的 39 名员工信息、合照、宣传画及价目表整合为一个美观的静态 HTML 页面。适合作为店铺的线上展示门户，供顾客预览员工、了解服务和价格。

页面采用 **复古幻想风格**，配色温暖柔和，适合 FF14 RP 店的氛围。

## 🚀 使用方法

1. 用浏览器直接打开 `index.html` 即可浏览。
2. 点击员工卡片可展开查看完整介绍。
3. 点击合照/宣传画/菜单原图可弹出放大查看。

### 本地预览

直接在浏览器中双击 `index.html` 即可，无需服务器。

## 🛠️ 页面功能

| 模块 | 说明 |
|------|------|
| 🏪 店铺头部 | 店名、地址、营业时间、游客群、简介 |
| 👥 员工介绍 | 39 名员工卡片网格展示，含头像、角色名、游戏 ID、爱好、介绍 |
| 📸 合照集 | 4 张集体合照画廊 |
| 🖼️ 宣传画 | 2 张宣传海报展示 |
| 📋 价目表 | 服务项目表格 + 小游戏列表 + 附加规则说明 + 菜单原图 |

---

## 📝 数据说明

- 员工数据来源于 `data/staff.md`，按照原始顺序拆分后与图片序号严格对应（第 1 名员工 → `card_01.png`，以此类推）。
- 员工共 39 人，卡片图片共 39 张。
- 所有卡片图片已统一转换为 PNG 格式。

---

## ⚠️ 注意事项

- 页面为纯静态 HTML，依赖本地图片资源。如需部署到线上，请确保图片路径正确。
- 如果在本地浏览器打开时图片不显示，请检查 `index.html` 与 `images/` 目录的相对位置关系（见上方文件结构）。
- 拍照（`.jpeg`）文件未转换格式，浏览器均可正常显示。
- 员工信息中可能包含颜文字、特殊 Unicode 字符，所有主流浏览器均可正常渲染。
